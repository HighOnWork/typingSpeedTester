# Typing Speed Tester (TST)
A Monkey Type clone built with Python.

## Prerequisites
Before running the application, ensure you have **Python 3.7+** installed.

### Linux Users
_Tkinter_ is usually bundled with Python on Windows and macOS. On Linux, you may need to install it manually via your package manager:

```
sudo apt-get install python3-tk
```

## Installation

### Clone the repository:
```
git clone https://github.com/HighOnWork/typingSpeedTester.git
```

### Install dependencies:
You will need _customtkinter_ for the UI and _requests_ to fetch random words from the API.

```
pip install customtkinter requests
```
Alternatively, if pip isn't in your path:

```
python -m pip install customtkinter requests
```

Focus Input: Start typing to begin the timer automatically.
